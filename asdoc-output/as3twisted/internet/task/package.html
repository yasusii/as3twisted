public function deferLater(delay:Number, closure:Function, ... args):Deferred

Call the given function after a certain period of time has passed.

Parameters

delay:Number — The number of seconds to wait before calling the function.
 
closure:Function — The function to call after the delay.
 
... args — The arguments to pass the function.

Returns
Deferred — A deferred that fires with the result of the callable when the specified time has elapsed.
