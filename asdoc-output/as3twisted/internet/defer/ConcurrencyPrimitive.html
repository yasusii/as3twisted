run()method 
public function run(func:Function, ... args):Deferred

This method takes a function as its first argument and any number of other positional arguments. When the lock or semaphore is acquired, the function will be invoked with those arguments.

Parameters

func:Function — The function which will be invoked when the lock or semaphore is acquired. This may return a Deferred; if it does, the lock or semaphore won't be released until that Deferred fires.
 
... args — The arguments to pass the function.

Returns
Deferred — Deferred of function result.
